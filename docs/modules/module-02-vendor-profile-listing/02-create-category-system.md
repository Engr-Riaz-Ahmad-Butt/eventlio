# Task 02 - Create Category System

## Goal
Allow vendors to be classified into service categories so clients can discover them more easily.

## Scope
- Create category entities such as `bridal salon`, `makeup artist`, or future service types
- Support one or many categories per vendor depending on final marketplace rules
- Add category management logic for selection and retrieval
- Prepare category data for marketplace filters and admin management

## Deliverables
- Database schema for categories and vendor-category relations
- Seed strategy or starter categories list
- Backend APIs or services for category selection
- Category retrieval utilities for frontend use

## Acceptance Criteria
- Categories can be created and retrieved reliably
- Vendors can be assigned to categories
- Category data can be reused by onboarding, profiles, and marketplace filters
- The schema supports future admin category CRUD
