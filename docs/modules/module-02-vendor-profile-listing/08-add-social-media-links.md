# Task 08 - Add Social Media Links

## Goal
Give vendors a way to connect trusted public social profiles to their business page.

## Scope
- Add fields for Instagram, Facebook, and future social links if needed
- Validate URL format and platform rules
- Render social links cleanly in edit and public profile views
- Prevent broken or malformed links from being saved

## Deliverables
- Social link fields in the data model
- Validation logic for supported platforms
- UI for adding and updating social links
- Public rendering rules for social buttons or links

## Acceptance Criteria
- Vendors can add valid Instagram and Facebook links
- Invalid URLs are rejected with clear feedback
- Saved links are displayed correctly on the public profile
- Empty social fields do not break the UI
