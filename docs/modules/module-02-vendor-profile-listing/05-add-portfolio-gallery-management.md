# Task 05 - Add Portfolio Gallery Management

## Goal
Enable vendors to showcase their work through a curated gallery.

## Scope
- Add portfolio item creation and deletion
- Support captions, ordering, and cover image selection if needed
- Show upload progress and gallery state in the UI
- Keep gallery data connected to the vendor profile

## Deliverables
- Portfolio gallery database relation
- Portfolio management UI
- Backend endpoints or actions for CRUD operations
- Ordered rendering support for public display

## Acceptance Criteria
- Vendors can add and remove portfolio images
- Gallery items stay linked to the correct vendor
- Optional captions or ordering fields persist correctly
- Portfolio content can be displayed publicly later without redesign
