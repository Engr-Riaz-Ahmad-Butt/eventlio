# Task 09 - Generate Public Vendor URLs

## Goal
Create clean and shareable public URLs for vendor business pages.

## Scope
- Define URL strategy using slug, vendor name, or another stable identifier
- Ensure public URLs are unique
- Handle slug collisions and updates safely
- Connect vendor URLs to public profile routing in the frontend

## Deliverables
- Slug or public URL field in the vendor profile model
- Slug generation and uniqueness logic
- Routing plan for public vendor pages
- Redirect or update strategy if vendor names change

## Acceptance Criteria
- Every public vendor profile has a unique URL
- URL generation is stable and predictable
- Name collisions do not break routing
- Public profile pages can resolve vendor data from the URL
