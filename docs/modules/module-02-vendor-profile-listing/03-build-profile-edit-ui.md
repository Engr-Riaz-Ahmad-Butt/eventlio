# Task 03 - Build Profile Edit UI

## Goal
Provide vendors with a structured interface to create and edit their business profile.

## Scope
- Build profile settings screens or sections
- Add fields for business name, description, contact details, and profile visuals
- Support save states, validation, and preview-friendly layout
- Keep the experience responsive on desktop and mobile

## Deliverables
- Vendor profile edit page or dashboard section
- Reusable form components for profile fields
- Save and update integration with backend
- User feedback for loading, success, and error states

## Acceptance Criteria
- Vendors can open and edit their business profile
- Required fields are validated before save
- Saved changes persist correctly and rehydrate on reload
- The UI is clean and usable across screen sizes
