# Task 07 - Build Password Reset Flow

## Goal
Allow users to recover account access securely when they forget their password.

## Scope
- Add forgot-password entry point
- Trigger password reset instructions
- Build reset-password screen
- Validate reset token or provider flow
- Confirm success and return user to login

## Deliverables
- Forgot-password form
- Reset-password form
- Reset token handling or provider integration
- Success and failure states

## Acceptance Criteria
- Users can request a password reset from the login flow
- Reset links or tokens work correctly
- Expired or invalid reset attempts are handled safely
- Users can sign in with the new password after completion
