# Task 03 - Build Signup & Login UI

## Goal
Create polished authentication screens for account creation and sign-in.

## Scope
- Build signup page
- Build login page
- Add form fields for email, password, and required identity data
- Add loading, success, and error states
- Keep the UI responsive for desktop and mobile

## Deliverables
- Signup screen
- Login screen
- Shared auth form components
- Connected form submission flow

## Acceptance Criteria
- Users can open dedicated signup and login pages
- Forms validate required fields before submission
- Errors are displayed clearly to the user
- Successful authentication redirects the user correctly
