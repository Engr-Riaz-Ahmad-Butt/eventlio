# Module 01 - Authentication & Onboarding

## Objective
Allow vendors and clients to securely register, log in, and set up their profiles with role-based access.

## Features
- Vendor registration
- Client registration
- Login / logout
- Forgot password
- Email verification
- OTP verification
- Role-based authentication
- Profile setup wizard
- Session management
